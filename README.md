## ğŸ¯ Project Overview

This project performs an in-depth analysis of financial loan data to uncover patterns, trends, and insights related to loan applications, approvals, and borrower characteristics. The analysis includes statistical exploration, data cleaning, feature engineering, and comprehensive visualizations.

## ğŸ“ Dataset

- **Source**: `financial_loan.csv`
- **Description**: Contains loan application data including borrower information, loan amounts, interest rates, income levels, and approval status
- **Key Features**: Loan amount, income, application type, loan status, interest rate, loan date, employment type

## ğŸ” Analysis Performed

### 1. **Data Exploration**
   - Dataset shape, structure, and data types
   - Missing value assessment
   - Initial data quality checks

### 2. **Descriptive Statistics**
   - Summary statistics for numerical columns
   - Distribution analysis of loan amounts and income
   - Loan status distribution

### 3. **Advanced Filtering & Segmentation**
   - High-value loan analysis (>$10,000)
   - Top income earner segments
   - Approval rate calculations

### 4. **Group Analysis**
   - Loan metrics by purpose/employment type
   - Age bracket segmentation and analysis
   - Default rate analysis

### 5. **Feature Engineering**
   - Debt-to-income ratio calculation
   - Correlation analysis
   - Age bracket categorization

### 6. **Outlier Detection**
   - IQR (Interquartile Range) method
   - Identification of anomalous loan amounts
   - Statistical bounds calculation
